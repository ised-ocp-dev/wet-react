# Lightbox

Using the `reverseSource` and `reverseQuote` props it is possible to right-justify the source or the entire block quote, respectively

## Default

default quote format

<Canvas>
  <Story
    name="Default"
    args={{
      children: 'Yeah, I said it, and you can quote me on that!',
      reverseQuote: false,
      reverseSource: false,
      footer: 'quote from man quoted',
      cite: 'journal',
    }}
  >
    {QuoteTemplate.bind({})}
  </Story>
</Canvas>

## reverseSource

If this boolean is set to true, the footer and citation will be right-justified

<Canvas>
  <Story
    name="reverseQuote"
    args={{
      children: 'Yeah, I said it, and you can quote me on that!',
      reverseQuote: false,
      reverseSource: true,
      footer: 'quote from man quoted',
      cite: 'journal',
    }}
  >
    {QuoteTemplate.bind({})}
  </Story>
</Canvas>

## reverseQuote

If this boolean is set to true, the entire block quote (quote, footer, citation) will be right-justified

<Canvas>
  <Story
    name="reverseSource"

      footer: 'quote from man quoted',
      cite: 'journal',
    }}
  >
    {QuoteTemplate.bind({})}
  </Story>
</Canvas>
