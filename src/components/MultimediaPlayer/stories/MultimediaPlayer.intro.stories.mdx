# MultimediaPlayer

Provides an accessible multimedia player for embedding video and audio into web pages.

## Example

<Canvas>
  <MultimediaPlayer
    sources={[
      {
        type: 'mp4',
        source: puppies,
      },
    ]}
    cuePoints={[
      { name: 'End of Intro', time: '15s' },
      { name: '', time: '00:30' },
    ]}
    figCaption={<p>Example Video</p>}
  />
</Canvas>
