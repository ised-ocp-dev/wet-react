# PrettyPrint

## Line Numbers

Use the `lineNums` attribute to add line numbers to code. Note you need to put your code in an ordered list `<ol>`

## Example

<Canvas>
  <Story name="Line Numbers" args={{ lineNums: true }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Custom Offset

Set the `lineNums` attribute to an integer to add line numbers starting at that many lines

## Example

<Canvas>
  <Story name="Custom Offset" args={{ lineNums: 2 }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Whole Page

Use the `wholePage` and `lineNums` attributes to add line numbers to all `<pre>` and `<code>` elements in the page

## Example

<Canvas>
  <PrettyPrint wholePage lineNums />
  <PrettyPrint>
    <ol>
      <li>Hey, this code was numbered!</li>
    </ol>
  </PrettyPrint>
</Canvas>
