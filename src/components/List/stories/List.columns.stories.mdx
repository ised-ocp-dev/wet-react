# List

## Column Count

Use `colCount` to break your list into 2, 3 or 4 columns

<Canvas>
  <Story name="Count" args={{ colCount: 3 }}>
    {ListTemplate.bind({})}
  </Story>
</Canvas>

## Size

Use `columnSize` to set the size of the columns to `xxs` `xs` `sm` `md` `lg` `xl`

<Canvas>
  <Story name="Size" args={{ colCount: 2, columnSize: 'xs' }}>
    {ListTemplate.bind({})}
  </Story>
</Canvas>

## Default to Maximum

When the list is too wide for the browser, it will default to the maximum number of columns that will fit

set `colCountDefaultMax` to false to make the list default to 1 column instead (not recommended)

<Canvas>
  <Story name="Default" args={{ colCount: 4, colCountDefaultMax: true }}>
    {ListTemplate.bind({})}
  </Story>
</Canvas>
